# San Francisco-Street-Tree-Analysis
This project analyzes publicly available data from (DataSF.gov) to provide a clear, interactive overview of the city’s street tree population. The dashboard reveals that trees overwhelmingly dominate the plant type category, with over 140,000 trees compared to only a few hundred landscaping features.
Data Source: SF Street Tree Inventory (DataSF.gov) 
Dashboard Link: https://bit.ly/Shaun-Tree-Dashboard
